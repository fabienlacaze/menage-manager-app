1. Configuration
Collez les URLs iCal depuis Airbnb et Booking. Ajoutez vos prestataires avec leur pourcentage (total = 100%).

Cliquez GENERER pour telecharger les calendriers et attribuer les menages automatiquement.

Fonctionnalites
Persistance : modifications sauvegardees automatiquement.
WhatsApp : envoyez le planning a chaque prestataire.
Calendrier : vue mensuelle avec menages et reservations.
Statistiques : graphiques de repartition par prestataire.
Historique : les menages passes sont archives.
Notifications : alertes 2 jours avant arrivee/depart.
Lien prestataire : partagez un lien read-only.
