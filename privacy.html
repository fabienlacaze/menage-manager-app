Politique de confidentialité

Dernière mise à jour : 23 mars 2026

1. Introduction

Cleaning Manager ("l'Application") est éditée par Fabidou. Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons vos données personnelles.

2. Données collectées

Nous collectons uniquement les données nécessaires au fonctionnement de l'Application :

  • Email : pour la création de compte et l'authentification
  • Données de planning : URLs de calendriers iCal (Airbnb, Booking), noms et numéros de téléphone des prestataires de ménage, plannings générés
  • Données de paiement : gérées exclusivement par Stripe. Nous ne stockons aucune information bancaire.

3. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Gérer votre compte et vous authentifier
  • Générer et sauvegarder vos plannings de ménage
  • Permettre la communication avec vos prestataires via WhatsApp
  • Traiter les paiements d'abonnement Premium via Stripe

4. Stockage et sécurité

Vos données sont stockées sur les serveurs de Supabase (hébergé par AWS, région EU Frankfurt). L'authentification utilise le chiffrement bcrypt. Toutes les communications sont chiffrées via HTTPS/TLS.

5. Partage des données

Nous ne vendons, ne louons et ne partageons jamais vos données personnelles avec des tiers.

Les services techniques suivants sont utilisés uniquement pour le fonctionnement de l'Application. Ils n'ont pas accès à vos données personnelles en dehors de leur rôle technique :

  • Supabase : hébergement technique de la base de données (vos données restent chiffrées et ne sont pas exploitées par Supabase)
  • Stripe : traitement sécurisé des paiements (nous ne stockons aucune donnée bancaire)
  • Google AdSense : affichage de publicités anonymes dans la version gratuite (aucune donnée personnelle n'est transmise à Google par notre Application)

6. Publicités

La version gratuite de l'Application affiche des publicités fournies par Google AdSense. Google peut collecter des données anonymisées pour personnaliser les annonces. Les utilisateurs Premium ne voient aucune publicité.

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Accès : consulter vos données personnelles
  • Rectification : modifier vos données inexactes
  • Suppression : demander l'effacement de vos données
  • Portabilité : exporter vos données (format CSV disponible dans l'app)
  • Opposition : vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : fabien65400@hotmail.fr

8. Cookies et stockage local

L'Application utilise le stockage local (localStorage) du navigateur pour sauvegarder vos préférences (langue, thème) et mettre en cache vos données pour un accès hors-ligne. Aucun cookie de suivi n'est utilisé par l'Application elle-même.

9. Conservation des données

Vos données sont conservées tant que votre compte est actif. En cas de suppression de compte, toutes vos données sont effacées sous 30 jours.

10. Modifications

Nous nous réservons le droit de modifier cette politique. Les utilisateurs seront informés de tout changement significatif.
